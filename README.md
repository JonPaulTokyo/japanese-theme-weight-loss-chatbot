# 🌸 影の語り部 | Japanese Weight Loss Chatbot

An ancient Japanese-themed **local weight loss mentor** powered by Ollama + Streamlit.

![Demo](https://i.imgur.com/placeholder.png) <!-- You can add a screenshot later -->

## Features
- Beautiful ancient Japanese aesthetic with falling sakura petals
- Specialized in **sustainable, healthy weight loss**
- Combines bushido discipline, Zen wisdom, and modern science
- Fully private & works offline
- Real-time streaming responses

Tech StackOllama (local LLM)
Streamlit (beautiful UI)
Fully offline capable

## How to Run Locally

```bash
# 1. Make sure Ollama is running
ollama run gemma3:4b

# 2. Run the chatbot
cd japanese-theme-weight-loss-chatbot
streamlit run app.py
